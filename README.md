# ☕ Barista Latte Art Portfolio

A beautiful, bilingual (English/Deutsch) portfolio website showcasing latte art creations. Built with pure HTML, CSS, and JavaScript.

## 🎨 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Bilingual Support**: Toggle between English and German with smooth language switching
- **Coffee-Themed Styling**: Custom color palette inspired by coffee tones
- **Gallery Display**: Showcase your latte art creations in an elegant card layout
- **No Dependencies**: Pure HTML, CSS, and JavaScript - no frameworks required

## 🚀 Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/your-username/latte_art_portfolio.git
cd latte_art_portfolio
```

2. Open `index.html` in your web browser to view the portfolio locally

### Customization

Edit the `index.html` file to:
- Replace the placeholder profile image with your actual photo
- Update gallery images with your latte art photos
- Modify the content text and descriptions
- Adjust colors in the CSS `:root` variables if desired
